# Tic-Tac-Toe Game

A classic Tic-Tac-Toe game built with React and TypeScript as part of learning React fundamentals.

## Features

- Interactive Tic-Tac-Toe gameplay
- Built with React and TypeScript
- Styled with Bootstrap
- Vite for fast development and building

## Technologies Used

- React
- TypeScript
- Bootstrap
- Vite

## Getting Started

### Prerequisites

- Node.js installed on your machine

### Installation

1. Clone the repository:
```bash
git clone git@github.com:bachok11/tic-tac-toe.git
cd tic-tac-toe
```

2. Navigate to the react-fundamentals directory:
```bash
cd react-fundamentals
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

## Project Structure

- `src/components/tic-tac-toe/Game.tsx` - Main game component
- `src/App.tsx` - Root application component
- `src/main.tsx` - Application entry point
